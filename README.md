## **Installation**

**Clone this reop:**

```shell
git clone https://github.com/del-luna/choco_python.git
cd choco_python
```

**Dependencies:**

All dependencies for defining the environment are provided in `requirements.txt`

### Start Server
```cmd
python start_flask.py
```
